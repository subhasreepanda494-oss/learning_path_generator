#  AI Learning Path Generator

## Description

The **AI Learning Path Generator** is a simple project that creates a structured learning roadmap for any skill using Python.
Users can enter a skill (like Python, AI, Data Science), and the program generates a step-by-step plan from beginner to advanced level.

---

##  Features

*  Beginner to Advanced roadmap
*  Structured learning path
*  Free learning resources
*  Estimated time for completion
*  Useful tips for success

---

## Tech Stack

* Python
* Jupyter Notebook / Google Colab

---

##  How to Run

1. Open the notebook in Google Colab
2. Run all cells
3. Enter a skill (e.g., Python, Machine Learning)
4. Get your personalized learning roadmap
